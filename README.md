A conversational AI chatbot built with Streamlit, LangChain, and Groq LLM.

🚀 Live Demo

https://app-genai-chatbot-aditya13.streamlit.app/

📂 Source Code

https://github.com/imaditya1313/streamlit-GenAI-Chatbot

✨ Features
💬 Interactive chat interface
⚡ Fast responses powered by Groq
🧠 Conversation history
🔐 Secure API key management using Streamlit Secrets

Python
Streamlit
LangChain
Groq
python-dotenv
📦 Installation
git clone https://github.com/imaditya1313/streamlit-GenAI-Chatbot.git
cd streamlit-GenAI-Chatbot
pip install -r requirements.txt
streamlit run Chatbot.py
